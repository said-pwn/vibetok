# Проверка работы приложения

## Быстрая проверка

### 1. Запустите приложение
```bash
npm run dev
```

### 2. Откройте браузер
Перейдите на `http://localhost:5173` (или другой порт, который покажет Vite)

### 3. Проверьте консоль
Нажмите `F12` и проверьте вкладку **Console** на наличие ошибок

### 4. Проверьте основные функции

#### ✅ Регистрация/Авторизация
- Перейдите на `/register` или `/login`
- Зарегистрируйтесь или войдите
- Должна работать авторизация через Firebase

#### ✅ Главная страница
- После авторизации вы должны попасть на `/`
- Должны отображаться видео (если есть)
- Должны работать кнопки в хедере

#### ✅ Загрузка видео
- Перейдите на `/upload`
- Выберите видео файл
- Заполните описание и хештеги
- Нажмите "Опубликовать"
- Видео должно появиться на главной странице

#### ✅ Лайки и комментарии
- На главной странице нажмите на лайк
- Нажмите на комментарии
- Добавьте комментарий
- Проверьте реакции на комментарии

#### ✅ Профиль
- Нажмите на иконку профиля в хедере
- Должен открыться ваш профиль
- Проверьте статистику

#### ✅ Новые функции
- **Тренды:** Нажмите на иконку трендов в хедере
- **Live:** Перейдите на `/live`
- **Сообщения:** Перейдите на `/messages`
- **Дуэты:** Нажмите на кнопку "Дуэт" под видео

## Если что-то не работает

### Проблема: Приложение не запускается
**Решение:**
1. Проверьте, что установлены зависимости: `npm install`
2. Проверьте версию Node.js (должна быть 16+)
3. Проверьте консоль на ошибки

### Проблема: Ошибка авторизации
**Решение:**
1. Проверьте конфигурацию Firebase в `src/config/firebase.js`
2. Проверьте, что Authentication включен в Firebase Console
3. Проверьте правила Firestore

### Проблема: Видео не загружаются
**Решение:**
1. Проверьте консоль на ошибки
2. Проверьте правила Firestore для коллекции `videos`
3. Убедитесь, что вы авторизованы

### Проблема: Комментарии не работают
**Решение:**
1. Проверьте правила Firestore для коллекции `comments`
2. Проверьте консоль на ошибки
3. Убедитесь, что вы авторизованы

### Проблема: Реакции не работают
**Решение:**
1. Проверьте правила Firestore для обновления комментариев
2. Проверьте консоль на ошибки
3. Убедитесь, что комментарии имеют поле `reactions`

### Проблема: Сообщения не работают
**Решение:**
1. Проверьте, что созданы коллекции `conversations` и `messages`
2. Проверьте правила Firestore для этих коллекций
3. Создайте индексы в Firestore (если требуется)

## Проверка Firebase

### 1. Проверьте конфигурацию
Откройте `src/config/firebase.js` и убедитесь, что все данные правильные:
- `apiKey`
- `authDomain`
- `projectId`
- `storageBucket`
- `messagingSenderId`
- `appId`

### 2. Проверьте Firebase Console
1. Откройте [Firebase Console](https://console.firebase.google.com/)
2. Выберите ваш проект
3. Проверьте:
   - **Authentication** → включен Email/Password
   - **Firestore Database** → создана база данных
   - **Storage** → создан Storage (если используется)

### 3. Проверьте правила
1. Откройте **Firestore Database** → **Rules**
2. Вставьте правила из файла `firestore.rules`
3. Нажмите **Publish**

## Отладка

### Включите подробное логирование
В консоли браузера вы увидите:
- Ошибки загрузки данных
- Ошибки Firebase
- Ошибки сети

### Проверьте Network
1. Откройте DevTools (`F12`)
2. Перейдите на вкладку **Network**
3. Попробуйте выполнить действие (например, загрузить видео)
4. Проверьте запросы к Firebase
5. Проверьте статус ответов (должны быть 200, не 403 или 400)

2. Укажите, какая функция не работает
3. Проверьте, что все шаги выше выполнены
